# MARGON

## Description

Très belle demeure du XVIe siècle à Arcisses (commune géocodée depuis "Margon", code postal 28400),
composée de 2 parcelles formant un terrain paysagé d'environ 1873m2 (annonce : ~2000m2).

La commune "Margon" est résolue automatiquement vers "Arcisses" (INSEE 28236) par le géocodeur.
Le DPE en base locale correspond à l'adresse "2 La Gentillere, 28400 Arcisses" (197.1m2, DPE D/GES B).
